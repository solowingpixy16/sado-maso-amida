# sado-maso-amida
サド-マゾ診断あみだくじ

## What is this? (これは何ですか？)
This is a joke repository for a browser game released on April Fools' Day, 2026.

これは、2026年のエイプリルフールネタとして公開されたブラウザゲーム用の、ジョークリポジトリです。

## What is the game like? (ゲームはどんな感じですか？)
The game purports to tell your fortune to determine whether you are a sadist or a masochist, but that is a lie. The game always claims that you are a masochist.
The game is hosted on [GitHub Pages](https://solowingpixy16.github.io/sado-maso-amida/).

このゲームでは、あなたがサディストかマゾヒストかを占っているかのように見せかけていますが、それは嘘です。常にマゾヒストであるという結果しか出ません。
ちなみにこのゲームは[GitHub Pages](https://solowingpixy16.github.io/sado-maso-amida/)上で公開されています。
